KI Web Component Games

This page demonstrates embedding games using only the custom tag and a single loader script.

**Instructions:** Use Arrows/A/D to Move, **Space** to Fire, **P** to Pause.

Space Invaders (ki-games-invaders)

The loader script detected this tag and automatically loaded **games/ki-games-invaders.js**.
